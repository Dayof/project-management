\subsection{Descrição Detalhada}
Domínio de Data. 

Este domínio só permite entradas nos formatos (01-\/31)/(01-\/12)/(2016-\/2050), (01-\/31)-\/(01-\/12)-\/(2016-\/2050), (01-\/31).(01-\/12).(2016-\/2050), (01-\/31) (01-\/12) (2016-\/2050). Ele é utilizado nas entidades para data de início de projeto/fase e data de término de projeto/fase.
